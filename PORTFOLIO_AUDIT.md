## Weak Project Removal Rules
A project is removed if it cannot answer all four:
1. What expensive problem did it solve?
2. What hard decision did you make?
3. What measurable outcome changed?
4. Why should a founder trust you with bigger scope after seeing this?

No answer = no showcase slot.
